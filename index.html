Projects

Letsplayy

Letsplayy is a WeChat Mini Program to organize, find, and easily share events with others. As roughly 34% of China's web traffic goes through WeChat, there is a clear need for an application that can bring people together on this platform. After pitching the idea, it was selected to be developed during the Le Wagon bootcamp. I acted as the lead developer and project manager, working with three other team members, to design and build this WeChat Mini Program along with the database and API that powers it. If you would like to learn more about the features of Letsplayy, you can find the code on my github: front end (Developed in WeChat's IDE) and back end (a RESTful API built with Ruby on Rails) or try it for free by scanning the QR code or by searching Letsplayy on WeChat.

Etienne Mister Cocktail

This Ruby on Rails application is a website where users can create, share, and discover cocktail recipes. Users can upload their own images of the cocktails they post and add ingredients imported from TheCocktailDB API. See the code on github (Ruby on Rails WebApp) or see the website in action at https://etienne-mister-cocktail.herokuapp.com.

Airlumni

Airlumni is another WeChat Mini Program. With this application, people associated with bootcamps would be able to post their profiles and share their services for hire in this AirBnB inspired platform. If you are interested in more details, feel free to check the code out: front end (WeChat IDE) and back end (Ruby on Rails API).
