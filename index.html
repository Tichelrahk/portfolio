Letsplayy

Letsplayy is a WeChat Mini Program platform to organize events and easily share them with others. As roughly 34% of China's web traffic goes through WeChat, it was clear to me that there is a lot of room for a program like this in the market. It's available for free by scanning this QR code or by searching Letsplayy on WeChat. I pitched this idea to my bootcamp and in a 2 week sprint, my team and I (four people) planned, designed, programmed, and developed this WeChat Mini Program along with the database and API that powers it. For this project, I took on the roles of lead developer and project manager. Take a closer look on my github: front end (Developed in WeChat's IDE) and back end (a RESTful API built with Ruby on Rails).
